# LibreWolf

- Settings>Privacy & Security
    - Select "Enable HTTPS-Only Mode in all windows"
- Settings>LibreWolf
    - Uncheck "Enable ResistFingerprinting"
        - Uncheck "Enable letterboxing"
        - Uncheck "Silently block canvas access requests"
    - Check "Enable WebGL"
